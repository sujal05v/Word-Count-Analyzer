# 📄 Word Count Analyzer (Python)
A Python program that reads a text file and counts the frequency of each word.
# 💡 Overview
This project was built as part of my Python learning journey.
It demonstrates file handling and the use of dictionaries to analyze and count word occurrences in a text file.
# ⚙️ Features
* 📂 Reads data from a file
* 🔤 Splits text into words
* 📊 Counts frequency of each word
* 🖨️ Displays word count output
# 🛠️ Technologies Used
* Python
# 📚 What I Learned
* File handling (read mode)
* Working with dictionaries
* String manipulation
* Looping through data
# 🎯 Purpose
This project was created to practice file operations and build a simple text analysis tool.
# 🔮 Future Improvements

* Make case-insensitive (Hello = hello)
